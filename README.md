# 🏦 VK RPG Bank Bot — Гайд для Банкиров

Добро пожаловать в систему управления виртуальным банком! Этот бот предназначен для контроля счетов персонажей в нашей ролевой игре.

## 👥 Роли и полномочия

1. **Игроки**:
   - Могут проверить свой счёт.
   - Могут снять деньги со своего счёта.
   - Могут посмотреть историю своих транзакций.
2. **Банкиры** (Администраторы):
   - Все права игроков + полное управление всеми счетами.
   - Начисление и списание средств у любых игроков.
   - Управление списком игроков.

---

## 📜 Команды для Игроков

- `/счёт` (или `/баланс`) — Показать текущий остаток на счёте и имя персонажа.
- `/снять [сумма] [причина]` — Снять деньги со своего счёта (например, за покупку в таверне).
- `/история` — Посмотреть 5 последних операций по своему счёту.
- `/регистрация [имя персонажа]` — Первичная регистрация в банке (без неё команды недоступны).
- `/помощь` — Список всех доступных команд.

---

## 👑 Команды для Банкиров

### 1. Начисление денег (форинтов - ft)
Поддерживаются гибкие форматы для массового пополнения:

- **Одному игроку**:
  `/начислить @упоминание 1000 за квест`
- **Нескольким (одна сумма)**:
  `/начислить @игрок1, @игрок2, @игрок3 500 групповая награда`
- **Нескольким (у каждого своя сумма)**:
  `/начислить @игрок1 200, @игрок2 450, @игрок3 150 разные заслуги`

### 2. Списание у игрока
- `/снять @упоминание 300 штраф` — Списывает `ft` со счёта указанного игрока.

### 3. Просмотр данных
- `/банк` — Выводит таблицу (ТОП-20) всех игроков, их персонажей и балансов.
- `/счёт @упоминание` — Быстро проверить баланс конкретного игрока.
- `/история @упоминание` — Просмотреть историю транзакций другого персонажа.

### 4. Управление аккаунтами
- `/изменитьперса @упоминание НовоеИмя` — Переименовать персонажа игрока.
- `/сделатьбанкиром @упоминание` — Назначить игрока помощником (выдать права Банкира).
- `/снятьбанкира @упоминание` — Забрать права Банкира (станет обычным игроком).
- `/удалить @упоминание` — Полностью удалить игрока из базы банка.

---

## ⚙️ Технические примечания
- Валюта: **ft** (форинты).
- Отрицательный баланс **запрещён**. Если у игрока не хватает денег для списания, бот выдаст ошибку.
- Бот работает по системе Long Polling и не требует внешней настройки вебхуков.
- Для корректной работы всегда используйте **@упоминание** игрока (просто нажмите на имя или выберите из списка ВК).
