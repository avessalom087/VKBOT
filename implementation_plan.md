# VK RPG Bank Bot — План реализации (Обновлённый)

## Текущее состояние
Проект находится на стадии завершения основной разработки. Все ключевые функции реализованы с учетом последних правок:
- ✅ **Валюта**: Переход на `ft` (форинты).
- ✅ **Роли**: Пользователи разделены на **Игроков** и **Банкиров**.
- ✅ **Команды**: Переименованы (`/счёт`, `/начислить`, `/снять`).
- ✅ **Массовые операции**: Банкиры могут начислять `ft` сразу нескольким игрокам разными способами.
- ✅ **Ограничения**: Игроки могут только проверять свой счёт и снимать деньги (самообслуживание).

## Инструкция по деплою (для пользователя)

### Шаг A: Переменные окружения на Render.com
Тебе нужно будет добавить следующие Environment Variables:
1. `VK_TOKEN`: Токен группы.
2. `ADMIN_VK_ID`: Твой ID (цифрами).
3. `FIREBASE_CREDENTIALS`: Весь текст из твоего `.json` файла (фигурные скобки и всё внутри).

### Шаг B: Запуск
Бот автоматически запустит HTTP-сервер на порту 8080 (или через переменную `PORT`).
- URL для UptimeRobot: `https://твой-сервис.onrender.com/health`
